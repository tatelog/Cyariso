# チャリ走 調査レポート

> 目的: チャリ走に忠実な類似ゲーム（Web/PWA）を開発するための事前調査。
> 本レポートはゲーム設計の判断材料として、コアメカニクス・操作性・レベル設計・
> 収益モデル・開発示唆を整理する。

## 1. 概要

「チャリ走（ちゃりそう）」は、スパイシーソフト株式会社が手がける**横スクロール・
オートランアクション**ゲームのシリーズ。ガラケー（フィーチャーフォン）時代の 2006 年に
登場して大ヒットし、その後スマートフォン・ニンテンドー 3DS・Wii U・PlayStation VR など
多数のプラットフォームへ展開。シリーズ累計ダウンロードは **3,000 万以上** とされる。

特徴は「**ジャンプだけ**」という極限までシンプルな操作と、迫りくる崖・段差を飛び越える
爽快感、そしてライバル「**サトウ**」から逃げ続けるというユーモラスな設定。

## 2. シリーズの変遷

| タイトル | 区分 | 特徴 |
|---|---|---|
| 元祖チャリ走 | ガラケー → スマホ復刻 | シリーズの原点。シンプルな逃走オートラン |
| チャリ走DX | iOS/Android/3DS | 自動スクロール＋ジャンプのみ。ステージ課金、コラボ（クレヨンしんちゃん／仮面ライダー鎧武 等） |
| チャリ走DX2 ギャラクシー | iOS/Android/3DS | DX の続編。12 星座モチーフの宇宙ステージ。「GALAXY TOUR」「GRAND PRIX」モード。変身チャリ（カエルチャリ＝高ジャンプ、ドリルチャリ＝岩破壊 等） |
| 跳びだせ!チャリ走3D | 3D | 3D グラフィックス化 |
| 超チャリ走 あつめて！超獣ハンター | 3DS | 収集要素強化 |
| チャリ走サバイバル | スマホ | 自転車カスタム＆対戦レース要素 |
| チャリ走〜シャル | ソーシャル版 | SNS 連携版 |

進化の方向性: **「ジャンプのみ」のコアは不変**のまま、(a) ステージ/世界観の拡張、
(b) 変身・乗り物バリエーションによる操作感の追加、(c) 収集・カスタム・対戦などの
メタ要素追加、で長寿命化している。

## 3. コアゲームメカニクス

- **オートラン**: プレイヤーは自動で前進し続ける。時間経過・進行で加速し難度が上がる。
- **ジャンプ操作のみ**: 画面のどこをタップしてもジャンプ。**空中で再タップ＝2 段ジャンプ**で
  高い段差・広い崖を越える（DX の根幹仕様）。
- **失敗即ゲームオーバー**: 崖への落下や障害物への接触など、一度のミスで終了（DX/DX2）。
- **ゴール制**: 決められた仕掛けが配置されたステージを走り、ゴールフラッグ通過でクリア。
- **追跡ライバル**: 後方からサトウが追ってくる演出（捕まる＝失敗のプレッシャー）。
- **変身/乗り物バリエーション（DX2）**: カエルチャリ（高ジャンプ）、ドリルチャリ
  （岩を破壊）など、乗り物で操作感・攻略が変化。

## 4. 操作性とゲームフィール

- **究極のシンプルさ**: 操作はジャンプ 1 つ。誰でも即プレイでき、ガラケー時代から
  スマホまで通用する設計。
- **爽快感**: 加速感＋ギリギリで崖を飛び越える緊張と達成感。2 段ジャンプの読み合いが肝。
- **短いプレイサイクル**: ミス即終了＋すぐリトライの「もう一回」ループでテンポが良い。
- **難易度カーブ**: 速度上昇と仕掛けの密度で自然に難化。

## 5. レベル/ステージ設計

- **ステージ制（DX 系）**: 手作りで仕掛けを配置したステージ群＋ゴール。コラボ IP の
  テーマステージで集客。
- **テーマ背景**: 街・宇宙（DX2 の 12 星座惑星）など、ステージごとに世界観を切替。
- **モード分け**: ステージ攻略型（GALAXY TOUR）と、走り続けてハイスコアを競う型
  （GRAND PRIX／サバイバル）の両輪。

## 6. 進行・収集・メタ要素

- 乗り物（変身チャリ）の収集・使い分け。
- カスタマイズ（サバイバルでは自転車カスタム）。
- オンラインランキング（GRAND PRIX）と対戦（サバイバル）。

## 7. ビジネス/収益モデル

- **基本プレイ無料＋ステージ課金**（DX）。
- **コラボ IP 戦略**: クレヨンしんちゃん、仮面ライダー鎧武などとのコラボステージで
  話題化・新規流入。
- ボリューム拡張（追加ステージ）で継続課金。

## 8. 類似ゲーム開発への示唆

再現すべき「核」:
1. **ジャンプのみ＋2 段ジャンプ**の即時操作（入力遅延ゼロ、画面どこでも反応）。
2. **加速していくオートラン**と崖/段差越えの爽快感。
3. **ミス→即リトライ**の高速ループ。
4. **追跡ライバル**による緊張演出。
5. **乗り物バリエーション**で操作感に幅（本作では自転車/バイク/車を選択可能に）。
6. **ステージ制＋エンドレス（ハイスコア）**の二本立て。

技術スタックの選択（本プロジェクト）:
- **HTML5 Canvas + バニラ JS**（ビルド不要・軽量・即実行）。
- **PWA（manifest + Service Worker）でオフライン対応**。
- 物理は等加速度の重力＋接地判定の自前実装で十分（外部エンジン不要）。
- アートは canvas 描画のオリジナルベクター調スプライト（後から実画像へ差替え可能に）。

MVP → フル実装スコープ:
- コアループ（オートラン＋ジャンプ＋崖判定＋ゲームオーバー）→ 乗り物選択 → ライバル →
  収集/スコア → ステージ/背景切替 → 進行保存（localStorage）→ PWA/オフライン。

差別化の余地:
- オリジナル世界観・キャラ（IP に依存しない）。
- 乗り物 3 種（自転車/バイク/車）の性能差による攻略の幅。
